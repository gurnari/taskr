```bash
Gestionnaire de tâches en CLI.

Usage: taskr <COMMAND>

Commands:
  add         Ajoute une tâche
  done        Valide une tâche
  list        Liste les tâches par ordre de priorité
  remove      Supprime une tâche
  clear-done  Supprime les tâches terminées
  help        Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
